# FitPact 💪

Squad fitness accountability app. Weekly check-ins, streak tracking, fines for missed weeks, banter feed.

## Deploy to Vercel (free, 5 min)

### Step 1 — Create GitHub repo
1. Go to github.com → sign in (or create free account)
2. Click **New repository**
3. Name it `fitpact`, set to **Public**, click **Create**

### Step 2 — Upload files
1. In your new repo, click **uploading an existing file**
2. Upload ALL files from this zip (keep folder structure)
3. Click **Commit changes**

### Step 3 — Deploy on Vercel
1. Go to vercel.com → sign up with GitHub
2. Click **Add New → Project**
3. Import your `fitpact` repo
4. Vercel auto-detects Vite — just click **Deploy**
5. Done! You get a free link like `fitpact.vercel.app`

### Step 4 — Share with squad
Share the Vercel URL in your WhatsApp group. Everyone opens it on their phone.

## Syncing data
Since everyone's on their own phone, use the **🔄 Sync** button:
- Admin exports a sync code after everyone checks in
- Share code in WhatsApp group
- Everyone imports to update their view

## Features
- ✅ Weekly check-ins with proof
- 📊 Leaderboard with streak tracking
- 💸 Fine ledger with paid/unpaid toggle
- 🗣️ Squad banter feed
- 📱 WhatsApp share button
- ⚙️ Group name, fine amount, deadline day customization
- 🔄 Data sync between devices
